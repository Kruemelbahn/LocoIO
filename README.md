# LocoIO

The LocoIO board is a DIY stationary interface to LocoNet for up to 16 input/output pins.
The original LocoIO design by [John Jabour](https://web.archive.org/web/20220405130232/http://locobuffer.com/) and was further developed by [Hans Deloof](https://locohdl.synology.me/).

![PCB from LocoIO topside](/Images/LocoIO.png)

Addional information about my experiences can be found in the [Documentation](https://github.com/Kruemelbahn/LocoIO/tree/main/Documentation) folder.
